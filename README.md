# Curso Cypress-basico-v2 do Walmyr

Sample project for the basic course of the TAT online course.

## Pre-requirements

It is required to have Node.js and npm installed to run this project.

> I used versions `v18.16.0` and `8.1.2` of Node.js and npm, respectively. I suggest you use the same or later versions.

## Installation

Run `npm install` (or `npm i` for the short version) to install the dev dependencies.

## Tests

You can run the project simulating a desktop or a mobile version.

### Desktop

Run `npm test` (or `npm t` for the short version) to run the test in headless mode.

Or, run `npm run cy:open` to open Cypress in interactive mode on a desktop viewport.

### Mobile

Run `npm run test:mobile` to run the test in headless mode on a mobile viewport.

Or, run `npm run cy:open` to open Cypress in interactive mode on a mobile viewport.
